## Features

- **Copy UUID** вАФ Right-click any entity with a stick (configurable) to copy its UUID to clipboard. [**Copy UUID**](https://github.com/Shiraishi-Sayuki/copy_uuid)
- **Dark Mode** вАФ Dark GUI shader overlay with multiple presets. [**Dark Mode Everywhere**](https://github.com/Buuz135/DarkModeEverywhere)
- **EatAnim** вАФ Adds eating/drinking animations for food and potions, with support for third-party mods. [**Eating Animation**](https://github.com/Theoness1/EatingAnimation)
- **Tooltip** вАФ Automatically wraps tooltips so they never go off-screen.
- **Play Time** вАФ Shows play time for worlds and servers on the selection screen. Configurable per-world/server: toggle on/off, position (Top Right/Left/Right/After Name/Behind Count), and color via the settings GUI.

## Commands
All commands are under `/sayukiutils`:

| Command | OP? | Description |
|---------|-----|-------------|
| `copyuuid toggle` | No | Toggle Copy UUID |
| `copyuuid set [item]` | No | Set trigger item |
| `copyuuid reset` | No | Reset set item |
