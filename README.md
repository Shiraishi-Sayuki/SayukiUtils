## 必要なMod:

**Fabric:**
- [**Fabric API**](https://modrinth.com/mod/fabric-api)
- [**Cloth Config API**](https://modrinth.com/mod/cloth-config)

**Forge:**
- [**Cloth Config API**](https://modrinth.com/mod/cloth-config)
  
## 機能

- **Copy UUID** — トリガーアイテム（デフォルト: 棒）でエンティティを右クリックするとUUIDをクリップボードにコピーできる。[**Copy UUID**](https://github.com/Shiraishi-Sayuki/copy_uuid)
- **ダークモード** — GUIを暗くするシェーダー。複数のプリセットから選べる。[**Dark Mode Everywhere**](https://github.com/Buuz135/DarkModeEverywhere)
- **EatAnim** — 食べ物や飲み物を食べるときにアニメーションが付くよ。他のModのアイテムにも対応。[**Eating Animation**](https://github.com/Theoness1/EatingAnimation)
- **Tooltip** — ツールチップが画面からはみ出ないようにいい感じに折り返す。
- **Play Time** — ワールド選択画面やサーバー選択画面にプレイ時間を表示する。表示位置（右上/左/右/名前の後/プレイヤー数の後）や色を設定画面から個別に設定可能。

## コマンド

全部 `/sayukiutils` から使える:

| コマンド | OPいる？ | 説明 |
|---------|----------|------|
| `copyuuid toggle` | いいえ | Copy UUID のON/OFF切り替え |
| `copyuuid set [アイテム]` | いいえ | トリガーアイテムを設定する |
| `copyuuid reset` | いいえ | トリガーアイテムをリセットする |
